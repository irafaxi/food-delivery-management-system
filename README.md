# Food Delivery Management System

## Description
A Java-based Food Delivery Management System developed using NetBeans.

## Features
- Manage customers (Regular & Golden Members)
- Manage restaurants and menu items
- Create and process orders
- Assign drivers and drones
- Calculate delivery costs

## Technologies
- Java
- Object-Oriented Programming (OOP)
- NetBeans

## How to Run
1. Open the project in NetBeans.
2. Build the project.
3. Run `FoodDeliverySystm.java`.
